原因？当然是不会写bat不会写判断啊

```
BaiduPCS-Go mkdir /apps/baidu_shurufa
```

然后windows用户要确保PATH配置为可以BaiduPCS-Go直接执行

或者新建BaiduPCS-Go.exe的快捷方式重命名为BaiduPCS-Go再扔进C:\Windows\并修改批处理文件加上".lnk"
